CSS stands for Cascading Style Sheets.


CSS can be written in 3 ways: 1. Inline 2. Internal 3. External


Inline CSS is applied directly to an HTML element using the "style" attribute.

Internal CSS is defined inside a <style> tag in the <head> section and applies styles globally.

External CSS is written in a separate .css file and linked to the HTML document using the <link> tag.
