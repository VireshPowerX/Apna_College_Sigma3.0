Bootstrap

Bootstrap is a free and open-source CSS framework directed at modern web development.

It contains CSS- and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites.

Bootstrap is a front-end framework that is used to create the visible part of the website.
