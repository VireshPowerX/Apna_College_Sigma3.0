Type can change (boolean to number) in JavaScript.
JavaScript is dynamically typed, meaning variable types can change.
Strings are written in "" or '' and can contain numbers, words, or be empty.
An empty string "" is treated as 0, while a space " " is treated as 1.
let name = "Tony"; name[0] is 'T', name[3] is 'y', name[name.length-1] is 'y'. name.length is 4.
name[name.length] is undefined because the index 4 does not exist.
"Tony" has positions 0,1,2,3 but total length is 4 (name.length).
Concatenation: "Tony" + " " + "Stark" = "Tony Stark" or "Tony" + 1 = "Tony1".
Keywords: true, false, null can be assigned to variables, but undefined is not a keyword.
let name = null; We can define it afterwards.
